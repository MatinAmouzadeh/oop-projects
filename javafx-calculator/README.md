# JavaFX Calculator

## Project Overview

This project is a simple calculator application built with **Java** and **JavaFX**.

The goal of this project is to practice object-oriented programming, graphical user interface development, event handling, and building a simple desktop application with JavaFX.

---

## Features

- Performs basic mathematical operations
- Receives user input
- Displays calculation results
- Simple JavaFX user interface
- Event handling practice
- Class design practice in Java

---

## Technologies Used

- Java
- JavaFX
- Object-Oriented Programming
- Event Handling
- Desktop Application Development

---

## Educational Concepts

This project was developed to practice the following concepts:

- Building a user interface with JavaFX
- Handling button click events
- Working with classes and objects
- Separating application logic from the user interface
- Implementing basic mathematical operations
- Practicing desktop application development with Java

---

## Project Status

This project was developed as a JavaFX and object-oriented programming practice project.

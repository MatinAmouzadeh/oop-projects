# OOP Projects

## Overview

This repository contains a collection of my practice projects developed while learning **Object-Oriented Programming with Java** and desktop application development using **JavaFX**.

The goal of these projects is to practice OOP concepts, class design, event handling, user interface development, and building small but useful applications with Java.

---

## Projects

### JavaFX Calculator

A simple calculator application built with Java and JavaFX.

### JavaFX Paint App

A simple paint application built with Java and JavaFX to practice graphics and user event handling.

### Improved Java Game

An improved version of an educational game originally developed by my instructor. The original source and instructor’s repository link will be credited in the project README.

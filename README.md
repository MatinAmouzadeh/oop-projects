The goal of this repository is to show my learning progress in OOP concepts, class design, project structure, JavaFX, Swing / AWT, and small educational desktop applications.

| `improved-java-game/` | An improved educational Java / OOP Brick Breaker-style game with a map designer tool | Java, Swing, AWT, OOP |

---

### About Improved Java Game

The `improved-java-game/` project is based on an educational Java / OOP project by **GholamAli Irani** and **AsenaTechAcademy**.

In this project, the original educational game was kept and improved in the `game/` module with features such as threaded execution, new levels, multi-ball gameplay, paddle size increase, and additional gameplay features.

This project was not written from scratch by me, but it was modified, extended, and organized by me for GitHub publication.

Original sources:

- https://github.com/AsenaTechAcademy/Irani-OOP-in-Java-gGame-Paranoid
- https://github.com/AsenaTechAcademy/Irani-OOP-in-Java-gGames-Paranoid-MapDesigner
- https://github.com/GholamaliIrani

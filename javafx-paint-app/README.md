# JavaFX Paint App

## Project Overview

This project is a simple paint application built with **Java** and **JavaFX**.

The goal of this project is to practice object-oriented programming, graphical user interface development, mouse event handling, and building an interactive desktop application with JavaFX.

---

## Features

- Draws on the screen using the mouse
- Handles mouse events
- Simple JavaFX user interface
- Java class design practice
- Interactive desktop application practice
- Clear canvas functionality

---

## Technologies Used

- Java
- JavaFX
- Object-Oriented Programming
- Event Handling
- Mouse Events
- Desktop Application Development

---

## Educational Concepts

This project was developed to practice the following concepts:

- Building a user interface with JavaFX
- Handling mouse events
- Working with Canvas or JavaFX graphical components
- Designing classes and methods
- Building a simple interactive application
- Separating application logic from the user interface

---

## Project Status

This project was developed as a JavaFX and object-oriented programming practice project.
